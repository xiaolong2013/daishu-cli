### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `daishu` 命令了。

- `daishu init webpack-bundl myWebpack`
- `daishu config get`
- `daishu config set type orgs`
- `daishu config set registry webpack-bundle`

- `daishu config set type users`
- `daishu config set registry xiaolong2013`

### 发布

开发完成后，即可发布至 npm.
